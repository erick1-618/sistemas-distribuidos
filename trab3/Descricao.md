# Trabalho 3

Implementar uma calculadora com o padrão de client e servidor RPC, onde o cliente solicita um serviço ao servidor e o servidor retorna uma resposta ao cliente.

[Explicação em Vídeo](https://youtu.be/QFBvENjX5jU)

### Requisitos para entrega:
1. O servidor deve oferecer os serviços das 4 operações aritméticas (adicão, subtração, divisão e multiplicação).

### Como executar
Rodar o servidor
```
python trab3/server.py
```
Rodar o cliente
```
python trab3/client.py
```

### Ferramentas
- Python, utilizando a implementação de RPC do pacote xmlrpc
